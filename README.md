# wanTooLs

Utility tools for web application developers!

## Features

- Directory Tree Editor
- CSS Transition Swatches
- Pointer Event Playground
- Keyboard Event Playground
- URI Encoding
- QR code Generator

## Prerequisites

- Node.js
- pnpm

## Developing

```bash
pnpm install
pnpm run dev
```

## Tech Stack

- [SolidStart](https://start.solidjs.com/)
- [Park UI](https://park-ui.com/)
